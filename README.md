# Hospital Management System Database

## Overview
This project presents the database design and implementation of a Hospital Management System.

The system manages:

- Patients
- Doctors
- Nurses
- Staff
- Departments
- Appointments
- Prescriptions
- Medications
- Medical Records
- Bills
- Rooms

## Database Design

The project includes:

- Entity Relationship Diagram (ERD)
- Relational Database Schema
- SQL Database Implementation

## Main Features

- Patient management
- Appointment scheduling
- Prescription management
- Medication tracking
- Billing system
- Department management
- Medical record management
- Room admission management

## Files

- `ERD.png` → Entity Relationship Diagram
- `Database_Schema.png` → Relational Schema Diagram
- `DB PROJECT.sql` → SQL script for database creation

- SQL
